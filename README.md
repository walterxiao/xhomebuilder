# AI Code Repository

This repository contains projects and code snippets related to AI and other web development experiments.

## Floorplan Furniture Builder

A simple browser-based application for dragging furniture into a floorplan and arranging it visually.

### Run locally

1. Open `index.html` in your browser.
2. Drag a furniture item from the palette into the floorplan.
3. Move items around inside the room.
4. Click the × button to remove an item.

### Files

- `index.html` — app structure and UI.
- `style.css` — layout and furniture styling.
- `app.js` — drag/drop and arrangement logic.
